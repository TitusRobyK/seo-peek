Display settings

Meta Keywords:

Detects a keywords metatag, ignored by most search engines.

Pagination Directives:

Detects the HTML link elements rel="next" and rel="prev" to indicate the relationship between component URLs in a paginated series.

Mobile Directives:

Detects the Vary HTTP header which helps Googlebot discover mobile-optimized content faster, the rel-amphtml annotation to let Google and other platforms know about the AMP version of the page, and the rel-alternate-media annotation which helps Google's algorithms to understand the configuration of a mobile site.

International Directives:

Detects the rel-alternate-hreflang annotation which helps Google to serve the correct language or regional URL in search results.
